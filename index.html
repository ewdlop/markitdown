⚠️ Microsoft MarkItDown 靜態版限制

此瀏覽器版僅支援純文字與部分簡單格式。PDF、Office、圖片、音訊等進階格式需要於完整的 Microsoft MarkItDown Python/CLI 環境中執行。

若需完整轉換能力，請在本機安裝套件或使用命令列：markitdown your-file.pdf -o output.md

以下限制說明由 AI 生成，涵蓋瀏覽器版無法支援進階格式的原因：
  • 缺乏原生依賴支援：MarkItDown 的 PDF、Office、圖片、音訊轉換需要多個 C/C++ 套件，而 Pyodide 只能載入純 Python 或 WebAssembly 套件。
  • 瀏覽器沙箱限制：靜態頁面無法呼叫系統工具或存取作業系統資源，許多高階功能因此受限。
  • 資源與效能考量：即使轉成 WebAssembly，載入體積與初始化時間也過大，且仍有功能無法完整運作。
